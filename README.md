# AI Gesture Security Interface

An AI-powered real-time gesture interaction system that detects users, tracks hand movements, enables air-writing, and provides a visual security-style interface using computer vision.

This project combines **hand tracking, motion detection, and human interaction visualization** to create a touchless interface for surveillance and gesture-based interaction.

---

## Features

- Real-time camera feed
- AI hand detection
- Air writing using index finger
- Erase drawings using open hand gesture
- Motion detection system
- Human detection with bounding box
- Real-time status display (motion detected, writing mode, erase mode)

---

## Technologies Used

- **MediaPipe Tasks Vision**
- **Computer Vision (Hand & Pose Detection)**
- **JavaScript (ES Modules)**
- **HTML5 Canvas**
